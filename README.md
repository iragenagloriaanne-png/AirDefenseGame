# Air Defense Shooter 🎮

## Description

Air Defense Shooter is a Java desktop game where a player on the ground defends against enemy airplanes.

## Features

* Shooting in all directions using mouse
* Enemy planes with bombs and bullets
* Knife enemies dropped from planes
* Wave system (difficulty increases)
* Explosion animations
* Destructible environment (tree, house, bunker, stairs)
* Restart button
* Difficulty selection (Easy, Medium, Hard)

## Controls

* Left Arrow → Move left
* Right Arrow → Move right
* Mouse → Aim
* Mouse Click / Space → Shoot

## How to Run

Compile:

```bash
javac src/AirDefenseGame.java
```

Run:

```bash
java -cp src AirDefenseGame
```

## Technologies Used

* Java
* Java Swing
* Multithreading
* Object-Oriented Programming
